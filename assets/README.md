# Free2Talk Assets

This directory contains project assets including logos, icons, and images.

## Logo

- **logo.svg** - Primary logo (SVG format, scalable)
- Used in README.md and documentation

## Usage

### In Markdown
```markdown
![Free2Talk Logo](./assets/logo.svg)
```

### In HTML
```html
<img src="./assets/logo.svg" alt="Free2Talk Logo" width="120" height="120">
```

## Customization

Feel free to replace the logo with your own design. Recommended formats:
- SVG for scalability
- PNG with transparent background (recommended: 512x512px)

## Brand Colors

- **Primary Blue:** `#3B82F6`
- **Dark Blue:** `#1E40AF`
- **White:** `#FFFFFF`

You can update the logo colors in `logo.svg` to match your brand guidelines.
